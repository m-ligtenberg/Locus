# 🚀 iOS .ipa maken zonder GitHub - Ionic Appflow

Omdat je niet bij GitHub kunt door 2FA, gaan we **Ionic Appflow** gebruiken - een gratis online build service!

## ✅ Stap 1: Account aanmaken
1. Ga naar: **https://ionic.io/appflow**
2. Klik **"Get Started Free"**
3. Maak account aan met email (GEEN GitHub required!)
4. Kies **"Hobby Plan"** (gratis, 1 build/maand)

## 📦 Stap 2: Project uploaden
Je hebt al het perfecte bestand klaar: `travelbot-ios-build.zip` (147KB)

1. In Appflow dashboard: klik **"New App"**
2. Kies **"Upload ZIP file"** 
3. Upload: `/home/mich/Documents/GitHub/travelbot/travelbot-ipa-builder/travelbot-ios-build.zip`
4. App naam: **TravelBot**

## 🏗️ Stap 3: iOS Build starten
1. Ga naar **"Builds"** tab
2. Klik **"New Build"**
3. Kies **iOS**
4. Platform: **iOS**
5. Build Type: **Development** (geen certificaat nodig)
6. Klik **"Build"**

## ⏱️ Stap 4: Wachten (10-15 min)
- Build duurt ~15 minuten
- Je krijgt email wanneer klaar
- Download .ipa direct vanuit dashboard

## 📱 Stap 5: Installeren
Met je .ipa kun je:
- **AltStore sideloading** (gratis)
- **TestFlight** (als je Apple dev account hebt)
- **Direct via Xcode** (op Mac)

## 🆓 Waarom Appflow?
- ✅ Geen GitHub 2FA nodig
- ✅ 1 gratis build per maand  
- ✅ Professionele build service van Ionic
- ✅ Ondersteunt Capacitor perfect
- ✅ Directe .ipa download
